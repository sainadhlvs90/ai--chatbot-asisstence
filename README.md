# AI Chatbot using Streamlit and Gemini AI

## Overview

This project is an AI-powered chatbot built using Python, Streamlit, and Google's Gemini AI model. The chatbot provides intelligent conversational responses through a user-friendly web interface.

## Features

* Real-time AI conversations
* Multiple Gemini model support
* Chat history display
* Clean and responsive Streamlit interface
* Secure API key management
* Easy deployment on Streamlit Community Cloud

## Technologies Used

* Python
* Streamlit
* Google Gemini AI API
* Git & GitHub

ai-chatbot/

## Installation

1. Clone the repository

git clone https://github.com/your-username/ai-chatbot.git

2. Navigate to the project folder

cd ai-chatbot

3. Install dependencies

pip install -r requirements.txt

4. Add your Gemini API key

Create a secrets file and add:

GEMINI_API_KEY = "your_api_key"

5. Run the application

streamlit run app.py
